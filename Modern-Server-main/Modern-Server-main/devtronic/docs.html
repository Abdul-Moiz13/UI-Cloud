Introduction

Removing the complexity around cloud hosting allows you to scale your websites how you want with our managed hosting for WordPress and WooCommerce. The simple-to-use Managed Web Hosting features include advanced caches coupled with Breeze, a simplified Loruki cache, and Cloudflare Enterprise for faster performance & added security. They all come together to provide a seamless web hosting experience.

Install CLI

Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js v12 or higher

Install

Mac (Homebrew)

$ brew install loruki-cli

NPM

$ npm install loruki-cli

Yarn

$ yarn install loruki-cli
